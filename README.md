# Introduction

Git: Widely adopted version control system

Feature: Fast | Disconnected | Powerful yet easy | Branching | Pull Requests

## Basic Git Commands

1. Configuring git

    ```
    git config --global user.name "<name>"
    git config --global user.email "<email>"
    ```

2. Getting help in git

    ```
    $ git help <verb>
    $ man git-<verb>
    $ git add -h (or --help)
